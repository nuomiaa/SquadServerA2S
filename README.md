# SquadServerA2S

用于 Steam Squad Server 的 A2S 服务

## 附言（2024/10/12 00:23）
该项目至此已归档保存，不再维护。

## 安装
```shell
npm install
```

## 运行
```shell
node main.js VP_IS=false VP_BLOCK=false
```

## 参数
| 名称 | 选项 | 解释 |
| :--: | :--: | :--: |
| VP_IS | true / false | 是否启用虚拟玩家人数 |
| VP_BLOCK | true / false | 是否屏蔽海外访问 |
